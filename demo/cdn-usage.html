🗺 MeowTileKit — 三种引用方式

方式一：<script> 标签（最简单）

<script src="https://unpkg.com/meow-tile-kit@0.1.0/dist/index.global.js"></script>
<script>
  const { wgs84ToGcj02, OSMSource, latLngToTile } = window.MeowTileKit;

  const gcj = wgs84ToGcj02(39.9042, 116.4074);
  console.log('GCJ-02:', gcj);

  const src = new window.MeowTileKit.OSMSource();
  console.log(src.getTileUrl({ x: 123, y: 456, z: 13 }));
</script>

⭐ 推荐方式 — 无需构建工具，直接打开 HTML 就能用

方式二：ES Module import

<script type="module">
  import { wgs84ToGcj02, AMapSource } from "https://unpkg.com/meow-tile-kit@0.1.0/dist/index.js";

  const gcj = wgs84ToGcj02(39.9042, 116.4074);
  const src = new AMapSource({ style: "satellite" });
  console.log(src.getTileUrl({ x: 123, y: 456, z: 13 }));
</script>

用于支持 type="module" 的现代浏览器

方式三：npm + 构建工具（生产项目推荐）

npm install meow-tile-kit
import { OSMSource, AMapSource, wgs84ToGcj02, latLngToTile, isInChina } from 'meow-tile-kit';

// 配合 Leaflet
const src = new AMapSource({ style: 'satellite' });
L.tileLayer((c) => src.getTileUrl(c)).addTo(map);

// 坐标转换
const gcj = wgs84ToGcj02(39.9, 116.4);
console.log('中国境内?', isInChina(39.9, 116.4));
